- 条件判断（if / elif / else）
- 循环（for / while / break / continue）

</details>

<details>
<summary><b>Stage 3：数据容器</b></summary>

- 列表（增删改查、推导式、二维列表）
- 元组（解包、多返回值）
- 字典（增删改查、遍历、嵌套）

<summary><b>Stage 4：函数</b></summary>

- *args / **kwargs
- lambda、map / filter / reduce
- 递归（阶乘、斐波那契）

</details>

<details>

- 封装、`@property`
- 运算符重载（`__add__`、`__str__` 等）
